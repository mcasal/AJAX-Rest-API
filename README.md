# Rest API con AJAX: Chuck Norris jokes.
Creación de una página web donde se hace una llamada AJAX a una API con chistes de Chuck Norris.

## Objetivos
* Hacer una llamada a una Rest API a través de jQuery Ajax.
* Crear una interfície gráfica acorde con el contenido.
* Realizar una petición a la API utilizando la función fetch.
